      API reference¶

      Datasets¶

      datasets_available() Example datasets that can be used with the library.

      Load¶

      load_csv(filepath_or_folder[, …]) Load timeseries data from a csv file, or group of files in a folder, into a DataFrame.
      load_campbell_scientific(filepath_or_folder) Load timeseries data from Campbell Scientific CR1000 formatted file, or group of files in a folder, into a DataFrame.
      load_excel(filepath_or_folder[, …]) Load timeseries data from an Excel file, or group of files in a folder, into a DataFrame.
      load_brightdata(dataset, lat, long, nearest) Retrieve timeseries datasets available from the brightdata platform.

      Analyse¶

      concurrent_coverage(ref, target, averaging_prd) Accepts ref and target data and returns the coverage of concurrent data.
      monthly_means(data[, return_data, …]) Plots means for calendar months in a timeseries plot.
      momm(data, date_from, date_to) Calculates and returns long term reference speed.
      distribution(var1_series, var2_series[, …]) Accepts 2 series of same/different variables and computes the distribution of first variable with respect to the bins of another variable.
      distribution_by_wind_speed(wspd[, return_data]) Accepts 2 series of same/different variables and computes the distribution of first variable with respect to the bins of another variable.
      distribution_by_dir_sector(var_series, …) Accepts a series of a variable and wind direction.
      freq_table(var_series, direction_series[, …]) Accepts a variable series and direction series and computes a frequency table of percentages.
      time_continuity_gaps(data) Returns the start and end timestamps of missing data periods.
      coverage(data[, period, aggregation_method]) Get the data coverage over the period specified.
      basic_stats(data) Gives basic statistical measures of the data, the DataFrame returned includes the following columns
      twelve_by_24(var_series[, …]) Accepts a variable series and returns 12x24 (months x hours) table for the variable.

      average_data_by_period(data, period[, …]) Averages the data by the time period specified by period.
      adjust_slope_offset(wspd, current_slope, …) Adjust a wind speed that already has a slope and offset applied with a new slope and offset.
      scale_wind_speed(spd, scale_factor) Scales wind speed by the scale_factor
      offset_wind_direction(wdir, offset) Add/ subtract offset from wind direction.
