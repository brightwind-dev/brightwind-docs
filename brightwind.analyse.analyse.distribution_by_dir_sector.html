    brightwind.analyse.analyse.distribution_by_dir_sector¶

    brightwind.analyse.analyse.distribution_by_dir_sector(var_series, direction_series, sectors=12, aggregation_method='%frequency', direction_bin_array=None, direction_bin_labels=None)[source]¶

    Accepts a series of a variable and wind direction. Computes the distribution of first variable with respect to wind direction sectors

    Parameters:
    • var_series – Series of the variable whose distribution we need to find
    • direction_series – Series of wind directions between [0-360]
    • sectors – Number of sectors to bin direction to. The first sector is centered at 0 by default. To change that behaviour specify direction_bin_array
    • aggregation_method – Statistical method used to find distribution it can be mean, max, min, std, count, describe, a custom function, etc. Computes frequency in percentages by default
    • direction_bin_array – Optional, to change default behaviour of first sector centered at 0 assign an array of bins to this
    • direction_bin_labels – Optional, you can specify an array of labels to be used for the bins. uses string labels of the format ‘30-90’ by default
    Returns:

    A DataFrame/series with wind direction sector as row indexes and columns with statistics chosen by aggregation_method
