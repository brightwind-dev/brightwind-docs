    brightwind.analyse.analyse.freq_table¶

    brightwind.analyse.analyse.freq_table(var_series, direction_series, var_bin_array=array([-0.5, 0.5, 1.5, 2.5, 3.5, 4.5, 5.5, 6.5, 7.5, 8.5, 9.5, 10.5, 11.5, 12.5, 13.5, 14.5, 15.5, 16.5, 17.5, 18.5, 19.5, 20.5, 21.5, 22.5, 23.5, 24.5, 25.5, 26.5, 27.5, 28.5, 29.5, 30.5, 31.5, 32.5, 33.5, 34.5, 35.5, 36.5, 37.5, 38.5, 39.5, 40.5]), sectors=12, var_bin_labels=None, direction_bin_array=None, direction_bin_labels=None, freq_as_percentage=True, return_data=False)[source]¶

    Accepts a variable series and direction series and computes a frequency table of percentages. Both variable and direction are binned

    Parameters:
    • var_series – Series of variable to be binned
    • direction_series – Series of wind directions between [0-360]
    • var_bin_array – Array of numbers where adjacent elements of array form a bin
    • sectors – Number of sectors to bin direction to. The first sector is centered at 0 by default. To change that behaviour specify direction_bin_array
    • var_bin_labels – Optional, an array of labels to use for variable bins
    • direction_bin_array – Optional, to change default behaviour of first sector centered at 0 assign an array of bins to this
    • direction_bin_labels – Optional, you can specify an array of labels to be used for the bins. uses string labels of the format ‘30-90’ by default
    • freq_as_percentage – Optional, True by default. Returns the frequency as percentages. To return just the count, set to False
    • return_data (bool) – Set to True if you want the data returned.
    Returns:

    A DataFrame with row indexes as variable bins and columns as wind direction bins.
