    brightwind.analyse.analyse.twelve_by_24¶

    brightwind.analyse.analyse.twelve_by_24(var_series, aggregation_method='mean', return_data=False, var_name='')[source]¶

    Accepts a variable series and returns 12x24 (months x hours) table for the variable. :param var_series: :param aggregation_method: ‘mean’ by default calculates mean of the variable passed. Can change it to

    ‘sum’, ‘std’, ‘min’, ‘max’, ‘percentile’ for sum, standard deviation, minimum, maximum, percentile
    of the variable respectively. Can also pass a function.
    Parameters:
    • return_data (bool) – Set to True if you want the data returned.
    • var_name (str) – Name and units of the variable to appear on the plot
    Returns:

    A DataFrame with hours as row labels and months as column labels.
