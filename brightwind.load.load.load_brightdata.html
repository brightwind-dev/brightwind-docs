    brightwind.load.load.load_brightdata¶

    brightwind.load.load.load_brightdata(dataset, lat, long, nearest, from_date=None, to_date=None)[source]¶

    Retrieve timeseries datasets available from the brightdata platform. Returns a list of Reanalysis objects in order of closest distance to the requested lat, long.

    Parameters:
    • dataset (str) – dataset type to be retrieved from brightdata e.g. merra2, era5.
    • lat (float) – latitude of your point of interest.
    • long (float) – longitude of your point of interest.
    • nearest (int) – the number of nearest nodes to your point of interest to retrieve. Currently only 1 to 4 is accepted.
    • from_date (str) – date from in ‘yyyy-mm-dd’ format.
    • to_date (str) – date to in ‘yyyy-mm-dd’ format.
    Returns:

    a list of Reanalysis objects in order of closest distance to the requested lat, long.
    Return type:

    List(Reanalysis)

    To use load_brightdata the BRIGHTDATA_USERNAME and BRIGHTDATA_PASSWORD environmental variables need to be set. In Windows this can be done by running the command prompt in Administrator mode and running:

    >>> setx BRIGHTDATA_USERNAME "username"
>>> setx BRIGHTDATA_PASSWORD "password"

    Example usage

    import brightwind as bw
nodes = bw.load_brightdata('era5', 53.4, -7.2, 4, '2018-01-01', '2018-10-01')
for node in nodes:
    print(node.data)
